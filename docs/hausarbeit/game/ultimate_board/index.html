Â§Contains the UltimateBoard struct

The UltimateBoard struct represents the board of the Ultimate Tic Tac Toe game. The board is represented as a 3x3 array of Board structs.

The UltimateBoard also contains the status of the game and the status of each board as well as the next board to play on.

The board contains a Zobrist hash used to store the evaluation of a Heuristic in a Transposition table.
